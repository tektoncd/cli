## tkn taskrun list

Lists taskruns in a namespace

### Synopsis

Lists taskruns in a namespace

```
tkn taskrun list
```

### Examples

```

# List all TaskRuns of Task name 'foo'
tkn taskrun list  foo -n bar

# List all taskruns in a namespaces 'foo'
tkn pr list -n foo \n",

```
