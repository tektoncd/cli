## tkn clustertriggerbinding delete

Delete ClusterTriggerBindings

***Aliases**: rm*

### Usage

```
tkn clustertriggerbinding delete
```

### Synopsis

Delete ClusterTriggerBindings

### Examples

Delete ClusterTriggerBindings with names 'foo' and 'bar'

    tkn clustertriggerbinding delete foo bar

or

    tkn ctb rm foo bar
