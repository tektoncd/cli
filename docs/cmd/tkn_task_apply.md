## tkn task apply

Create or update a task in a namespace

### Usage

```
tkn task apply
```

### Synopsis

Create or update a task in a namespace

### Examples

Create a task or update already existing task defined by foo.yaml in namespace 'bar':

    tkn task apply -f foo.yaml -n bar
