## tkn hub get pipeline

Get Pipeline by name, catalog and version

### Usage

```
tkn hub get pipeline
```

### Synopsis

Get Pipeline by name, catalog and version

### Examples


Get a Pipeline of name 'foo':

    tkn hub get pipeline foo

or

Get a Pipeline of name 'foo' of version '0.3':

    tkn hub get pipeline foo --version 0.3
