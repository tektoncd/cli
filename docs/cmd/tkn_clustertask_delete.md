## tkn clustertask delete

Delete a clustertask resource in a cluster

***Aliases**: rm*

### Usage

```
tkn clustertask delete
```

### Synopsis

Delete a clustertask resource in a cluster

### Examples


# Delete a ClusterTask of name 'foo'
tkn clustertask delete foo

tkn ct rm foo
