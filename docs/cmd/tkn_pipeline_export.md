## tkn pipeline export

Export Pipeline

### Usage

```
tkn pipeline export
```

### Synopsis

Export Pipeline

### Examples

List all PipelineRuns of Pipeline 'foo':

    tkn pipelinerun list foo -n bar

List all PipelineRuns in a namespace 'foo':

    tkn pr list -n foo
