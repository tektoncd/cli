## tkn pipelinerun describe

Describe a PipelineRun in a namespace

***Aliases**: desc*

### Usage

```
tkn pipelinerun describe
```

### Synopsis

Describe a PipelineRun in a namespace

### Examples

Describe a PipelineRun of name 'foo' in namespace 'bar':

    tkn pipelinerun describe foo -n bar

or

    tkn pr desc foo -n bar
