## tkn clustertask delete

Delete ClusterTasks in a cluster

***Aliases**: rm*

### Usage

```
tkn clustertask delete
```

### Synopsis

Delete ClusterTasks in a cluster

### Examples

Delete ClusterTasks with names 'foo' and 'bar':

    tkn clustertask delete foo bar

or

    tkn ct rm foo bar
